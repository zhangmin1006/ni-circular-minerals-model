# Streamlit Community Cloud installs THIS file at the repo root.
#
# The ONLINE app only reads the committed model outputs (CSV/JSON in
# model/outputs/) and renders them, so it needs just these three packages
# (+ their transitive numpy). It does NOT import mesa / scipy / networkx — those
# are only used by the model itself, which on cloud runs only if a "Run … now"
# button is clicked, and that never happens because the outputs are committed.
#
# Keeping the cloud build minimal avoids the C-extension build failures
# (scipy / mesa toolchain) that otherwise break the online deploy. The FULL
# model + experiment stack is in model/requirements.txt (used by CI and local
# runs). Upper bounds keep a future breaking major from silently breaking deploy.
streamlit>=1.49,<2       # st.dataframe(width="stretch") needs >= 1.49
pandas>=2.2,<3.1
matplotlib>=3.8,<4
